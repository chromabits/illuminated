Class RouteMapperTestCase

Class RouteMapperTestCase.

This test case attempts to efficiently find common errors in RouteMapper classes and route definitions.

Do note that these tests make big assumptions on how the code is structured. You might need to override or replace certain checks to meet your own project.

This test can be particularly slower than other since it performs some introspection on controllers.
